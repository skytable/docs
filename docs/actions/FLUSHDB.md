title: FLUSHDB
---

:::note About
**Time complexity**: O(n)  
**Accept type**:

- [AnyArray](../../protocol/data-types#any-array)

**Return type**:

- [Rcode 0](../../protocol/response-codes)
- [Rcode 5](../../protocol/response-codes)

**Syntax**:

- `FLUSHDB`
- `FLUSHDB <entity>`

:::

Removes all entries stored in the current table or in the provided entity
