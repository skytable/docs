:::note About
**Time complexity**: O(1)  
**Accept type**:

- [AnyArray](../../protocol/data-types#any-array)

**Return type**:

- [Integer](../../protocol/skyhash#unsigned-integers-)

**Syntax**:

- `DBSIZE`
- `DBSIZE <entity>`

:::

Check the number of entries stored in the current table or in the provided entity
