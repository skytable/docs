title: EXISTS
---
        

:::note About
**Time complexity**: O(n)  
**Accept type**:

- [AnyArray](../protocol/data-types.md#any-array)

**Return type**:

- [Integer](../protocol/skyhash.md#unsigned-integers-)

**Syntax**:

- `EXISTS <key1> <key2> ...`

:::

Check if 'n' keys exist in the current table. This will return the number of keys that exist
as an unsigned integer.
