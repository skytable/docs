Actions overview

Actions are like shell commands: they take arguments and do something! TerrabaseDB currently supports the following actions:

  • DEL
  • EXISTS
  • GET
  • HEYA
  • MGET
  • MSET
  • MUPDATE
  • SET
  • UPDATE
