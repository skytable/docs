---
        

:::note About
**Time complexity**: O(1)  
**Accept type**:

- [AnyArray](../protocol/data-types.md#any-array)

**Return type**:

- [Integer](../protocol/skyhash.md#unsigned-integers-)

**Syntax**:

- `DBSIZE`
- `DBSIZE <entity>`

:::

Check the number of entries stored in the current table or in the provided entity
