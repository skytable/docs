title: MGET
---
        

:::note About
**Time complexity**: O(n)  
**Accept type**:

- [AnyArray](../protocol/data-types.md#any-array)

**Return type**:

- [Typed Array](../protocol/data-types.md#typed-array)

**Syntax**:

- `MGET <key1> <key2> ...`

:::

Get the value of 'n' keys from the current table, if they exist
