:::note About
**Time complexity**: O(1)  
**Accept type**:

- [AnyArray](../protocol/data-types.md#any-array)

**Return type**:

- [Integer](../protocol/skyhash.md#unsigned-integers-)
- [Rcode 1](../protocol/response-codes.md)

**Syntax**:

- `KEYLEN <key>`

:::

Returns the length of the UTF-8 string, if it exists in the current table
