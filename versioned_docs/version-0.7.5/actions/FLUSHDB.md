title: FLUSHDB
---


:::note About
**Time complexity**: O(n)  
**Accept type**:

- [AnyArray](../protocol/data-types.md#any-array)

**Return type**:

- [Rcode 0](../protocol/response-codes.md)
- [Rcode 5](../protocol/response-codes.md)

**Syntax**:

- `FLUSHDB`
- `FLUSHDB <entity>`

:::

Removes all entries stored in the current table or in the provided entity
