---
id: sdel
title: SDEL
---
:::note About
**Time complexity**: O(n)  
**Arguments**: `SDEL <key1> <key2> ...`  
**Returns**: (Code: 0) if all keys were deleted, otherwise (Code: 1)  
:::
Delete all keys if all of the keys exist. Do note that if a single key doesn't exist, then a `Nil` code is returned.
