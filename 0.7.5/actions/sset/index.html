Version: 0.7.5

SSET

About

Time complexity: O(n)
Accept type:

  • AnyArray

Return type:

  • Rcode 0
  • Rcode 2
  • Rcode 5

Syntax:

  • SSET <key1> <value1> <key2> <value2> ...

Set all keys to the given values only if all of them don't exist in the current table
